# 通用后台非功能需求清单

## 1. 权限与安全

- 是否需要登录鉴权
- 是否需要角色权限隔离
- 是否涉及敏感数据脱敏

## 2. 审计与日志

- 是否需要关键操作日志
- 是否需要状态流转留痕
- 是否需要错误日志

## 3. 性能与稳定性

- 列表查询响应是否有要求
- 是否需要大数据量分页
- 是否要考虑批量操作稳定性

## 4. 可用性

- 是否需要移动端适配
- 是否需要失败重试或兜底提示
- 是否需要空状态、异常状态、无权限状态

## 5. 可维护性

- 是否需要模块化拆分
- 是否需要统一组件和样式规范
- 是否需要接口抽象层

## 6. 部署与兼容

- 使用端是 Web 还是多端
- 是否有浏览器兼容要求
- 是否有私有化部署或环境隔离要求
