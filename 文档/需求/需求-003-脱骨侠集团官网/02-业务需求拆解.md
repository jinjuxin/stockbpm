# 业务需求拆解

## 1. 拆解前提

本轮按以下确认结果继续：

- 品牌主体：脱骨侠，鸡爪食品集团
- 官网目标：品牌展示
- 整体调性：食品集团企业感 + 消费品牌年轻化
- 首页重点：产品/业务实力
- 语言版本：中文、越南语
- 使用端：Web

## 2. 业务场景

本次不是交易平台，也不是后台系统，而是集团官网。

核心场景包括：

- 外部访客快速认识集团品牌与实力
- 合作伙伴了解业务板块和合作基础
- 媒体与公众查看企业新闻和社会责任信息
- 求职者了解企业文化与招聘信息

## 3. 官网目标

- 建立脱骨侠集团统一、正式的品牌形象出口
- 让访客快速理解集团业务实力、品牌定位和发展方向
- 支撑新闻传播、社会责任展示和招聘入口
- 兼顾中国与越南语用户浏览需求

## 4. 系统边界

### 第一版要解决

- 集团品牌展示
- 集团介绍与品牌故事
- 业务板块展示

- 招聘入口展示

## 9. 核心浏览流程

### 主流程

1. 访客进入首页
2. 快速感知品牌形象与产品/业务实力
3. 根据兴趣进入集团介绍、业务板块、新闻、社会责任、招聘或联系我们
4. 进一步了解企业内容并形成品牌认知或合作/联系动作

### 典型分支流程

- 合作伙伴：首页 -> 业务板块 -> 联系我们
- 求职者：首页 -> 招聘 -> 联系我们
- 媒体或公众：首页 -> 新闻中心 / 社会责任 -> 集团介绍

## 10. 双语策略

- 默认提供中文和越南语两个版本
- 一级导航结构保持一致
- 同一页面原则上双语内容一一对应
- 若部分内容暂不具备越南语版本，后续需要明确空缺策略
