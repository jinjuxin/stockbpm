# 角色权限矩阵

## 角色清单

- 提交人
- 处理人
- 管理员

## 页面与操作矩阵

| 页面/模块 | 操作 | 提交人 | 处理人 | 管理员 |
| --- | --- | --- | --- | --- |
| 工单列表 | 查看自己相关工单 | 是 | 是 | 是 |
| 工单列表 | 查看全部工单 | 否 | 否 | 是 |
| 工单列表 | 搜索与筛选 | 是 | 是 | 是 |
| 新建工单 | 创建工单 | 是 | 否 | 是 |
| 工单详情 | 查看详情 | 是 | 是 | 是 |
| 工单详情 | 查看处理记录 | 是 | 是 | 是 |
| 工单详情 | 更新状态 | 否 | 是 | 是 |
| 工单详情 | 分派处理人 | 否 | 否 | 是 |
| 分派页 | 指定处理人 | 否 | 否 | 是 |
| 工单数据 | 查看自己提交的工单 | 是 | 否 | 是 |
| 工单数据 | 查看分派给自己的工单 | 否 | 是 | 是 |
| 工单数据 | 查看全部工单 | 否 | 否 | 是 |

## 当前结论

- 第一版权限模型较简单，按角色直接区分
- 暂不引入更细粒度的数据权限和菜单权限
- 若后续新增部门隔离、多组织架构，需要扩展权限模型
