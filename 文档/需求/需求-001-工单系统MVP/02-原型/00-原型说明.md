# 原型说明

## 1. 原型范围

本轮原型覆盖工单系统 MVP 的 4 个核心页面：

- 列表页
- 新建页
- 详情页
- 分派页

## 2. 页面清单

### 2.1 列表页

文件：
- `列表页.html`

用途：
- 查看工单列表
- 执行基础搜索与筛选
- 进入详情页
- 管理员进入分派页

展示字段：
- 工单编号
- 标题
- 优先级
- 提交人
- 处理人
- 状态
- 创建时间

筛选项：
- 标题关键词
- 状态
- 优先级
- 处理人

### 2.2 新建页

文件：
- `新建页.html`

用途：
- 提交人创建工单

字段：
- 标题
- 内容
- 优先级

只读字段：
- 提交人
- 初始状态
- 创建时间

### 2.3 详情页

文件：
- `详情页.html`

用途：
- 查看工单完整信息
- 展示当前状态与可执行动作

展示字段：
- 工单编号
- 标题
- 内容
- 提交人
- 处理人
- 优先级
- 创建时间
- 当前状态

### 2.4 分派页

文件：
- `分派页.html`

用途：
- 管理员为工单指定处理人

字段：
- 工单编号
- 当前状态
- 标题
- 当前处理人
- 处理人选择
- 分派备注

## 3. 角色差异

### 3.1 提交人

- 可创建工单
- 只看自己提交的工单
- 可查看详情
- 不可分派处理人
- 不可推进状态

### 3.2 处理人

- 只看分派给自己的工单
- 可查看详情
- 可按顺序推进状态
- 不可分派处理人

### 3.3 管理员

- 可查看全部工单
- 可分派处理人
- 可直接推进状态

## 4. 状态流转

固定顺序：

`待处理 -> 处理中 -> 已完成 -> 已关闭`

约束：
- 不允许跳级流转
- 已关闭后默认不可再编辑

## 5. 页面跳转

- 列表页 -> 新建页
- 列表页 -> 详情页
- 列表页 -> 分派页
- 新建页 -> 详情页
- 新建页 -> 列表页
- 详情页 -> 列表页
- 详情页 -> 分派页
- 分派页 -> 详情页

## 6. 本轮原型结论

当前原型已满足进入“原型确认”阶段的最小要求：

- 页面范围明确
- 字段明确
- 角色差异明确
- 关键跳转明确
- 核心状态流转明确
